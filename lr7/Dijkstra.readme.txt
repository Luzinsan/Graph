В первой строке входных данных задаётся количество вершин графа.
Далее в каждой последующей строке первым элементом выступает вершина графа.
Сразу же через пробел перечисляются смежные вершины графа в виде пар (вершина, вес). Это означает, что сначала выписывается номер вершины, с которой смежна текущая вершина, а сразу за ней идёт значение веса дуги, связывающей текущую вершину (первый элемент в строке) с номером только что напечатанной вершины. Строка должна обязательно заканчиваться двумя нулями, написанными через пробел, что обозначает конец записи смежных вершин для текущей вершины. 
Примерный образец:
6                               # количество вершин
1 2 4 3 9 4 3 0 0               #  1 - текущая вершина; 2, 3, 4 - смежные вершины; 4,9,3 - соответствующие веса дуг { (1, 2), (1, 3), (1, 4) }
2 3 3 5 2 0 0                 # 2 - текущая вершина; 3, 5     - смежные вершины; 3,2    - соответствующие веса дуг { (2, 3), (2, 5) }
3 6 2 0 0
4 5 3 0 0
5 6 2 0 0
6 0 0
